Projetos de Machine Learning

Sistema de Ranqueamento de Clientes para Cross-selling de Seguros

Objetivo:

Criar um algoritmo de ranqueamento de clientes propensos a compra de um seguro veícular, a partir de dados de pesquisa de interesse realizada com clientes antigos.

Execução:

A solução desenvolvida foi ranquear os clientes com base em um score de propensão de compra calculado através de um modelo de Machine Learning e apresentar esse ranking através de uma integração com Apps Script no Google Sheets.

Ferramentas utilizadas:

  • Python com pacotes de Machine Learning: sklearn e xgboost.
  • Algoritmos: Random Forest, XGboost, Naive Bayes, Regressão logística e KNN
  • Jupyter Notebook, Git, Github e Vscode.
  • Google Apps Script.
  • Técnicas de seleção de atributos e redução de dimensionalidade.
  • Render Cloud, Flask e Python API's.
  • Google Sheets

Resultados financeiros:

  • Realizando 18 mil ligações com base na lista ranqueada, incremento previsto de 176% na receita com seguro veicular, tendo como baseline a escolha aleatória de clientes.

    • Projeto concluído em 10/2023.

Previsão de Vendas de Rede de Lojas

Objetivo:

Criar um algoritmo de previsão de vendas para uma rede de lojas, e disponibilizar os resultados via smartphone

Execução:

Criação de um modelo de Machine Learning capaz de prever o faturamento das vendas de uma rede de lojas (Rossmann). Disponibilizando as previsões através de um bot do telegram, o bot recebe o número da loja e fornece a previsão de faturamento das próximas 6 semanas.

Ferramentas utilizadas:

  • Python com pacotes de Machine Learning: sklearn e xgboost.
  • Algoritmos: Random Forest, XGboost, Linear Regressor e Linear Regressor Regularized
  • Jupyter Notebook, Git e Github.
  • Técnicas de seleção de atributos e redução de dimensionalidade.
  • Render Cloud, Flask e Python API's.

Resultados financeiros:

  • O erro entre o nosso baseline (média de vendas) e o resultado real foi de 15%, contra apenas 2% do modelo desenvolvido.

    • Projeto concluído em 06/2023.
  • Saiba Mais (Github)

Projetos de Insights

Dashboard para identificação de imóveis para compra e futura locação.

Objetivo:

Responder a seguinte questão de negócio:
  1. Quais são os melhores bairros e tipos de imóveis a serem comprados para termos a maior rentabilidade?

Execução:

Respondi as questões através de uma análise exploratória de dados em Python. Para isto, construí um dashboard interativo acessível via navegador, que serve de base para a tomada de decisão do time de negócio. Mais tarde também inclui um dashboard em Power BI

Ferramentas utilizadas:

  • Python, Pandas, Numpy e Seaborn.
  • Vscode
  • Jupyter Notebook, Git e Github.
  • Mapas interativos com Plotly e Folium.
  • Streamlit Web App.
  • Streamlit Cloud.
  • Power BI.

Resultados financeiros:

Infelizmente devido a base de dados não possuir o valor de vendas do imóveis não pude quantificar o resultado financeiro trazido pelo projeto. Porém foi possivel apresentar diversos insights sobre regiões, bairros e tipos de imóveis que mais gerariam rentabilidade quando locados.
Projeto concluído em 01/2023.
